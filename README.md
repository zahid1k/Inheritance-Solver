# Inheritance-Solver

Inheritance Solver for family tree

#Input formats :

"CHILD A S B C D"  -> "A" and "S" are the parents of "B", "C", "D".
"CHILD A S B"      -> "A" and "S" are the parents of "B".
"MARRIED A B"      -> "A" and "B" are married.
"DEPARTED B"       -> "B" died.
"DECEASED A 600"   -> "A" died with 600 heritage.
